## Roadmap General
### Fase 1: MVP (Sprints 1-2)

**Objetivo:** Funcionalidades core de corte de peso

**Entregables:**
- 

Sistema de autenticación completo
- 

Dashboard básico con métricas
- 

Registro manual de alimentos
- 

Tracking de peso diario
- 

Calculadora básica de corte de peso
### Fase 2: Funcionalidades Avanzadas (Sprints 3-4)

**Objetivo:** Dashboard especializado y análisis avanzado

**Entregables:**
- 

Integración completa con Gemini 
- 

Análisis de alimentos por fotografía
- 

Sistema de alertas inteligentes
- 

Dashboard con métricas especializadas
### Fase 3: Optimización (Sprints 5-6)

**Objetivo:** PWA, funcionalidades offline, pulido final

**Entregables:**
- 

Progressive Web App completa
- 

Optimización de performance
- 

Testing completo
- 

Preparación para lanzamiento
